# OOPS
## Class 
Class are the bluprint of the object.
#### Syntax:
class *classname* {
    *type variable1;*
    *type variable2;*
    *type variable3; ...*
    *type function1(parameter-list)*{
    <pre>
        *body of the function1*
    </pre>
    }
}
## Object
Object is the instance of the class.
